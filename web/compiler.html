Building and Installing a Cross Compiler
   

Prerequisites

You need a native GNAT compiler on your system. Nowadays almost all Linux distributions contain the Ada enabled gcc (≥ 3.4). On MinGW there is a precompiled gcc-4.2.1 with Ada support. Please pick the -dw version as the -sjlj version is broken (as of Nov. 2007).

You need the GNU binutils and the avr-libc library. The first is available on any GNU mirror. Take at least version 2.16. The latter is available from the AVR libc home page. I use the release 1.4.5.

The gcc sources must be from the 4.2 branch of gcc. I did all my tests and modifications on gcc-4.2.1.

Get the latest patches from the AVR-Ada project. The patches are also included in the AVR-Ada source package in the directory patches/.

Building the compiler

Have a look at the shell scripts build-avr-ada-*.sh in the tools/build/ directory. It automates the build process and we use it exclusively for building our compilers and tools.
